# Material de Entrenamiento
## Olimpiadas Ecuatorianas de Astronomía y Astrofísica

Este repositorio contiene notas de las clases de los programas de Entrenamiento Básico y Avanzado de la Olimpiada Nacional de Astronomía.

Está distribuido en **áreas**.

1. Astronomía de Posición
2. Mecánica Celeste
3. Astrofísica Estelar
4. Cosmología
5. Instrumentación.
6. Galaxias.
7. Análisis de Datos.
8. Varios.

El repositorio irá llenándose de material progresivamente. También incluiremos lecturas en español para apoyar el proceso.
